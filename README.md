## 📂 Local Setup & Installation

Since this project uses Vanilla JavaScript and Tailwind via CDN, there is no complex build step required. 

**1. Clone the repository:**
`git clone https://github.com/yourusername/your-repo-name.git`

**2. Navigate to the directory:**
`cd your-repo-name`

**3. Run the project:**
Simply open `index.html` in your favorite modern web browser, or use an extension like VS Code Live Server for hot-reloading.
